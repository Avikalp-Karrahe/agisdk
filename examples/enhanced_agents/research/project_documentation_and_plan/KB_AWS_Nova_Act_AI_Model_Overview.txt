Amazon Nova Act is a research preview of an SDK and AI model designed for developers to build agents that can take actions in web browsers. <mcreference link="https://github.com/aws/nova-act" index="1">1</mcreference> <mcreference link="https://github.com/aws/nova-act/blob/main/README.md" index="4">4</mcreference> It allows developers to break down complex workflows into smaller, reliable commands, add details, call APIs, and intersperse direct browser manipulation using Python. <mcreference link="https://github.com/aws/nova-act" index="1">1</mcreference> <mcreference link="https://github.com/aws/nova-act/blob/main/README.md" index="4">4</mcreference>

Key points about Nova Act:
-   **Experimental SDK:** It's an early research preview and may make mistakes. Users are responsible for monitoring its use. <mcreference link="https://github.com/aws/nova-act" index="1">1</mcreference> <mcreference link="https://github.com/aws/nova-act/blob/main/README.md" index="4">4</mcreference>
-   **Prompt Injection Risk:** Nova Act might encounter unauthorized commands (prompt injections) on third-party websites, potentially causing errors or deviations from instructions. It's recommended to limit its operations to trusted websites. <mcreference link="https://github.com/aws/nova-act" index="1">1</mcreference> <mcreference link="https://github.com/aws/nova-act/blob/main/README.md" index="4">4</mcreference>
-   **API Key Security:** API keys should be kept confidential. If compromised, a new key can be obtained from `https://nova.amazon.com/act`. <mcreference link="https://github.com/aws/nova-act" index="1">1</mcreference> <mcreference link="https://github.com/aws/nova-act/blob/main/README.md" index="4">4</mcreference>
-   **Sensitive Information:** It's advised not to provide sensitive information like passwords directly to Nova Act. <mcreference link="https://github.com/aws/nova-act" index="1">1</mcreference> <mcreference link="https://github.com/aws/nova-act/blob/main/README.md" index="4">4</mcreference>
-   **Data Collection:** Interactions, prompts, and screenshots are collected to improve the service. Data deletion can be requested. <mcreference link="https://github.com/aws/nova-act" index="1">1</mcreference> <mcreference link="https://github.com/aws/nova-act/blob/main/README.md" index="4">4</mcreference>
-   **Availability:** Nova Act is a research preview, free to use with a daily request quota, and accessed via an amazon.com account, not directly integrated as an AWS product. <mcreference link="https://github.com/aws/nova-act/blob/main/FAQ.md" index="2">2</mcreference> It's currently available in the US, Canada, and Mexico. <mcreference link="https://github.com/aws/nova-act/blob/main/FAQ.md" index="2">2</mcreference>
-   **Functionality:** It supports browser automation, information extraction, and automated UI tasks. <mcreference link="https://github.com/aws/nova-act/blob/main/FAQ.md" index="2">2</mcreference> It does not yet support general computer use or integration with tools like LangChain. <mcreference link="https://github.com/aws/nova-act/blob/main/FAQ.md" index="2">2</mcreference>
-   **Technical Requirements:** Requires Python 3.10+ and works best with Google Chrome. <mcreference link="https://github.com/aws/nova-act/blob/main/README.md" index="4">4</mcreference> It supports English. <mcreference link="https://github.com/aws/nova-act/blob/main/README.md" index="4">4</mcreference>
